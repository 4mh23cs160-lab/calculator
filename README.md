# Calculator Project

A complete browser calculator app with UI, keyboard support, and expression evaluation.

## Files

- `index.html`: main page with calculator layout
- `styles.css`: UI styling and responsive design
- `script.js`: calculator logic (input handling, compute, clear, backspace, keyboard)

## Features

- Basic arithmetic: + - � �- Parentheses `()`
- Exponent `^`
- Percent `%`
- Sign toggle `±`- Decimal input
- Clear (`C`), backspace (?), equals (`=`)
- Keyboard support: digits, operators, Enter
- Safe expression check and error handling

## Run

Open `index.html` in your browser.
